# Daniel Foster — Portfolio Website

This is a simple static portfolio website designed for GitHub Pages.

## Files

- `index.html` — site structure and content
- `styles.css` — layout, colors, responsive styling
- `script.js` — mobile navigation and footer year
- `assets/` — put your resume PDF and project images here

## How to publish with GitHub Pages

1. Create a new GitHub repository.
2. Upload these files to the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Choose your main branch and root folder `/`.
6. Save.
7. Your site should publish at a GitHub Pages URL.

## Things to customize

- Replace the LinkedIn placeholder in `index.html`.
- Add your real resume as `assets/Daniel-Foster-Resume.pdf`.
- Replace `#` project links with GitHub repositories, PDFs, or detailed project pages.
- Add images or diagrams for each project.
- Add a custom domain later if desired.
